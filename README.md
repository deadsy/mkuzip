# FreeBSD mkuzip compiled for Linux and MacOS.

https://github.com/deadsy/mkuzip

## Linux Build

```
$ sudo apt install zlib1g-dev liblzma-dev libzstd-dev
$ make
```

## MacOS Build

```
$ brew install xz zstd
$ make
```
